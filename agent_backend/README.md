## Установка

1. Убедитесь, что у вас установлен Node.js версии 22 или выше. Вы можете проверить это, выполнив команду:

   ```bash
   node -v
   ```

2. Установите зависимости проекта, выполнив команду:

   ```bash
    npm install
   ```

3. Запустите проект, выполнив команду:

   ```bash
   npm start
   ```
